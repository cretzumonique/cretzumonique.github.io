# Monica Cretu — Computer Science ePortfolio

**Live site:** [cretzumonique.github.io](https://cretzumonique.github.io)

## About

This repository hosts my CS 499 Computer Science Capstone ePortfolio for the BS Computer Science program at Southern New Hampshire University. It showcases three enhancements applied to a single artifact — the Weight Tracker Android Application originally developed in CS 360 — across the three core areas of computer science: Software Engineering and Design, Algorithms and Data Structures, and Databases.

## Contents

- **Professional Self-Assessment** — reflection on the program and capstone experience
- **Code Review** — video walkthrough of the original artifact and planned enhancements
- **Enhancement One: Software Engineering and Design** — progress dashboard, dirty flag pattern, DiffUtil adapter, input validation, Javadoc
- **Enhancement Two: Algorithms and Data Structures** — Trend Analysis Engine with 5 algorithms (rolling average, trend detection, binary search, multi-criteria sorting, streak calculator)
- **Enhancement Three: Databases and Security** — SHA-256 password hashing, input validation layer, security audit documentation, hardened SQLite configuration, 30 JUnit unit tests

## Repository Structure

- `index.html` — the ePortfolio site (rendered via GitHub Pages)
- Source code files for each enhancement
- Project zip files containing the full Android Studio project for each milestone

## Technologies

Java, Android Studio, SQLite, JUnit
