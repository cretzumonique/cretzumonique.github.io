Code Review Highlights

Key findings from the code review that drove the enhancement plan:

Software Engineering: No architectural pattern (MVVM absent), business logic embedded in Activity classes, no input validation, orphaned SMS method.

Algorithms & Data Structures: Entries ordered by insertion ID rather than date, no sorting or filtering, no trend analysis or rolling average, O(n) full scan on every load with no optimization.

Databases: Critical plaintext password storage, inconsistent query safety (rawQuery with string concatenation), destructive onUpgrade strategy, no DAO abstraction layer.

Enhanced Artifacts

All three enhancements apply to the same artifact — the Weight Tracker Android Application originally developed in CS 360. Using one artifact across all three categories demonstrates the ability to iteratively improve a real system across multiple dimensions of computer science.

Enhancement One — Software Engineering and Design

Status: Complete — submitted for Milestone Two

Key enhancements:

Progress Dashboard Dirty Flag Pattern Input Validation DiffUtil Adapter SMS Goal Trigger Javadoc

Added a progress dashboard with ProgressBar, implemented the dirty flag pattern to eliminate redundant database reloads, added numeric range validation across all entry points, replaced notifyDataSetChanged with DiffUtil, wired the sendGoalReachedSms method to trigger automatically when goal is reached, and added comprehensive Javadoc to all classes and methods.

Enhancement Two — Algorithms and Data Structures

Status: Complete — submitted for Milestone Three

Key enhancements:

7-Day Rolling Average O(n) Trend Detection Binary Search O(log n) Multi-Criteria Sort O(n log n) Streak Calculator O(n)

Introduced a Trend Analysis Engine (WeightAnalytics, AnalyticsManager, TrendSummary) implementing five algorithms: sliding window rolling average, trend direction detection, binary search for date-range lookup, TimSort-based multi-criteria sorting, and best-streak calculator. Added sort Spinner to UI and analytics dashboard panel to MainActivity.
